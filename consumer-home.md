# 消费者端 - 首页原型

## 页面定位
用户打开小程序看到的第一页，承载商品浏览、搜索、分类、推荐等核心入口。

## 交互说明

| 区域 | 交互 |
|------|------|
| 搜索栏 | 点击进入搜索页，支持关键词/语音搜索 |
| 购物车图标 | 跳转购物车页，显示角标数量 |
| Banner轮播 | 自动轮播，点击跳转活动页 |
| 快捷入口 | 6个图标入口，点击跳转对应页面 |
| 商品卡片 | 点击进入商品详情页 |
| 底部导航 | Tab切换，当前页高亮 |

---

## 设计要点

1. **信息密度适中** - 首屏展示关键入口+4-6个商品
2. **视觉层次清晰** - 搜索→Banner→入口→商品，自然引导
3. **转化入口突出** - 购物车、特惠、兑换等入口醒目
4. **品牌感** - 简洁留白，突出商品图片
