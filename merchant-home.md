## 交互说明

| 区域 | 交互 |
|------|------|
| 订单统计卡片 | 点击跳转对应状态订单列表 |
| 发货按钮 | 弹出发货弹窗→填写快递单号→确认 |
| 联系客户 | 一键拨打电话 |
| 批量发货 | 勾选多个订单→批量填写快递单号 |
| 打印发货单 | 生成PDF发货单 |
| 通知项 | 点击展开详情 |

---

## 设计要点

1. **任务导向** - 待处理订单优先展示，减少点击
2. **操作便捷** - 核心操作按钮显眼，一键可达
3. **信息完整** - 订单卡片包含所有关键信息
4. **批量处理** - 支持批量操作提升效率
5. **配色克制** - 商家端偏重效率，减少装饰性元素
6. **状态明确** - 待处理用橙色警示，已发货用蓝色，完成用绿色
