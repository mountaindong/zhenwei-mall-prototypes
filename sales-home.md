## 交互说明

| 区域 | 交互 |
|------|------|
| 收益卡片 | 点击展开详细收益明细页 |
| 今日数据 | 点击跳转数据详情页 |
| 商品海报 | 选择商品→生成推广海报→保存/分享 |
| 推广链接 | 一键复制专属推广链接 |
| 分享小程序 | 调起微信分享，带销售员专属标识 |
| 我的团队 | 点击进入团队管理页 |

---

## 设计要点

1. **收益为王** - 核心数据最大最醒目，激发动力
2. **工具便捷** - 推广工具一键可达，降低操作成本
3. **数据可视化** - 趋势图直观展示收益变化
4. **团队激励** - 展示团队成员业绩，形成竞争氛围
5. **配色心理** - 金色/橙色强调收益，绿色表示增长
